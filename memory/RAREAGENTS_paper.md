# RAREAGENTS: Advancing Rare Disease Care through LLM-Empowered Multi-disciplinary Team

## Basic Information
- **Title**: RAREAGENTS: Advancing Rare Disease Care through LLM-Empowered Multi-disciplinary Team
- **Arxiv ID**: 2412.12475
- **Paper URL**: https://arxiv.org/abs/2412.12475
- **HTML Version**: https://arxiv.org/html/2412.12475v1
- **Date Found**: 2025-11-13

## Description
An innovative framework for enhancing diagnosis and treatment of rare diseases using LLM-empowered multi-disciplinary teams. It's the first LLM-driven multi-disciplinary team decision-support tool designed specifically for complex clinical contexts involving rare diseases.

## Key Features
- Integrates advanced Multidisciplinary Team (MDT) coordination
- Memory mechanisms and medical tools utilization
- Uses Llama-3.1-8B/70B as base models
- Patient-centered multi-disciplinary agent-based framework
- Outperforms GPT-4o and current agent frameworks
