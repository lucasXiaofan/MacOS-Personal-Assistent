## Key Contributions
- Novel multi-agent framework for medical decision-making
- **Automatically assigns collaboration structure** (solo or group) tailored to medical task complexity
- Mirrors real-world medical decision-making processes via dynamic collaboration
- First adaptive decision-making framework for LLMs in medical domain

## Evaluation
- Tested on multiple benchmarks: MedQA, MedMCQA, PubMedQA, DDXPlus, PMC-VQA, PathVQA, MedVidQA
- **Best performance in 5 out of 7 benchmarks** requiring multi-modal medical reasoning
- Shows excellent efficiency in adapting number of collaborating agents
